# Contact Form with JavaScript Validation

## 📝 Objective
Build a simple and responsive contact form that validates user input (name, email, and message) using JavaScript on the client side and provides user feedback messages.

## 💻 Tools Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- VS Code
- Chrome Browser

## 📁 Project Structure
```

/contact-form/
│
├── index.html         # Main HTML file with form and script
└── README.md          # Project documentation

```

## 📌 Features
- Clean and responsive contact form layout
- Client-side input validation:
  - Checks if name, email, and message are filled
  - Validates email format using RegEx
  - Ensures message length is adequate
- Displays real-time error messages and success feedback

## 🚀 How to Run the Project
1. Clone or download the repository.
2. Open `index.html` in your browser (you can use Live Server in VS Code for best experience).
3. Fill in the form and click **Send Message**.
4. Errors will show if inputs are invalid. A success message is shown if all inputs are valid.
